# WorkMate TikTok Ad Package

## Deliverables

- `workmate-tiktok-keyvisual.png`: generated 9:16 key visual for the ad.
- `workmate-tiktok-ad.html`: animated vertical preview for browser/screen recording.
- `workmate-app-demo-vertical.html`: animated app-screen demo showing a populated WorkMate marketplace.
- `workmate-app-demo-vertical.mp4`: exported 15-second vertical video for phone transfer and posting.
- `workmate-app-demo-vertical.pptx`: PowerPoint source used to export the MP4.
- `captions.srt`: 15-second subtitle file.
- `external-ai-prompts.md`: prompts for Runway, Pika, Canva, CapCut, or similar tools.

## Recommended Format

- Platform: TikTok, Instagram Reels, YouTube Shorts
- Canvas: 9:16 vertical
- Size: 1080 x 1920
- Duration: 15 seconds
- Tone: fast, clear, recruitment-focused
- Target:
  - Sydney hospitality employers
  - Working holiday visa holders and international students

## Manual Export Path

Because this environment could not run Node, Python, or ffmpeg reliably, the current practical export path is:

1. Open `workmate-tiktok-ad.html` in a browser.
2. Use browser fullscreen or a 9:16 recording area.
3. Screen-record 15 seconds.
4. Add the voiceover and captions from `captions.srt` in CapCut, Canva, or TikTok editor.

For a higher-end result, use `external-ai-prompts.md` in Runway/Pika/Canva and upload `workmate-tiktok-keyvisual.png` as the reference image.

## App Screen Demo Version

Use `workmate-app-demo-vertical.html` when you want the ad to show the app after many employers and students have registered.

It includes simulated screens for:

- Home activity dashboard
- Jobs feed
- Staff profiles
- DM conversation
- Employer applicant dashboard

Recommended use:

1. Open `workmate-app-demo-vertical.html`.
2. Record 15 seconds in a vertical 9:16 frame.
3. Add voiceover or music in CapCut/Canva/TikTok.
